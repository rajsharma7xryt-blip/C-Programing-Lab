# Pointer Playground

## 📌 Project Overview
Pointer Playground is a C programming project designed to demonstrate the fundamental concepts of pointers. It covers displaying memory addresses, dereferencing pointers, swapping two numbers using pointers, traversing arrays using pointers, and performing pointer arithmetic.

This project helps beginners understand how pointers work and how they are used for efficient memory management in C.

---

## 🎯 Objectives
- Understand pointer declaration and initialization.
- Display variable memory addresses.
- Access values using pointer dereferencing.
- Swap two numbers using pointers.
- Traverse an array using pointers.
- Demonstrate pointer arithmetic.
