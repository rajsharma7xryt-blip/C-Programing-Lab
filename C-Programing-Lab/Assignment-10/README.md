# 🏆 Assignment-10: LeetCode Array Challenge

## 📌 Objective

The objective of this assignment is to implement various array operations in C language to solve common array-based programming problems. The program demonstrates searching, sorting, frequency counting, duplicate removal, reversing arrays, and finding largest/smallest elements while analyzing the time complexity of each operation.

---

## 📝 Problem Statement

Develop a menu-driven **LeetCode Array Challenge** program that performs the following operations on an array:

- Find the Largest and Second Largest element
- Find the Smallest and Second Smallest element
- Reverse the array
- Remove duplicate elements
- Count the frequency of each element
- Perform Linear Search
- Sort the array in ascending order

The program should use **Arrays**, **Searching**, **Sorting**, and **Algorithm Analysis** concepts.

---

## 🛠️ Concepts Used

- Arrays
- Functions
- Loops
- Searching Algorithms
- Sorting Algorithms
- Frequency Counting
- Duplicate Removal
- Algorithm Analysis
- Menu-Driven Programming

---

## 📋 Algorithm

1. Start the program.
2. Input the size and elements of the array.
3. Display the main menu.
4. Read the user's choice.
5. Perform the selected operation:
   - Largest & Second Largest
   - Smallest & Second Smallest
   - Reverse Array
   - Remove Duplicates
   - Frequency Count
   - Linear Search
   - Sort Array
6. Display the result.
7. Repeat until the user chooses Exit.
8. Stop the program.

---

## ⏱️ Time Complexity

| Operation | Time Complexity |
|-----------|-----------------|
| Largest & Second Largest | O(n) |
| Smallest & Second Smallest | O(n) |
| Reverse Array | O(n) |
| Remove Duplicates | O(n²) |
| Frequency Count | O(n²) |
| Linear Search | O(n) |
| Bubble Sort | O(n²) |

---

## 💾 Space Complexity

**O(n)**

where **n** is the number of elements in the array.
