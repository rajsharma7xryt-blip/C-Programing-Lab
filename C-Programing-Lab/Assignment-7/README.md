# 🏥 Assignment-07: Hospital Patient Management System

## 📌 Objective
The objective of this assignment is to develop a menu-driven Hospital Patient Management System using C language. The program allows users to register, display, search, update, and delete patient records efficiently using structures and arrays.

---

## 📝 Problem Statement
Develop a Hospital Patient Management System that stores patient details such as Patient ID, Name, Age, Gender, Disease, and Contact Number. The application should provide the following functionalities:

- Register a new patient
- Display all patient records
- Search patient by Patient ID
- Update patient information
- Delete patient records

The program should be implemented using **Structures, Arrays, Strings, and Functions**.

---

## 🛠️ Concepts Used
- Structures
- Arrays
- Strings
- Functions
- Menu-Driven Programming
- Searching
- Updating Records
- Deleting Records

---

## 📋 Algorithm

1. Start the program.
2. Display the main menu.
3. Read the user's choice.
4. Perform the selected operation:
   - Add Patient
   - Display Patients
   - Search Patient
   - Update Patient
   - Delete Patient
5. Repeat until the user chooses Exit.
6. Stop the program.

---

## ⏱️ Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add Patient | O(1) |
| Display Patients | O(n) |
| Search Patient | O(n) |
| Update Patient | O(n) |
| Delete Patient | O(n) |

---

## 💾 Space Complexity

**O(n)**

where **n** is the number of patient records stored.
