# 💻 C Programming & Data Structures Lab Assignments

A collection of **10 C Programming & Data Structures Lab Assignments** covering fundamental programming concepts such as Arrays, Strings, Pointers, Structures, Dynamic Memory Allocation, Searching, Sorting, and Menu-Driven Applications.

This repository is designed to strengthen problem-solving skills and provide hands-on experience with core concepts of the C programming language.

| Assignment-05 | Dynamic Array Management | Dynamic Memory Allocation, Pointers |
