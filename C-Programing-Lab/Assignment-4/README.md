# Student Record Management System

## 📌 Project Overview
The Student Record Management System is a menu-driven C program that manages student records using **structures**, **arrays of structures**, **strings**, and **functions**. The system allows users to add, display, search, update, and delete student records efficiently.

This project is designed to strengthen the understanding of data structures and record management in C programming.

---

## 🎯 Objectives
- Learn the use of structures in C.
- Store multiple student records using arrays of structures.
- Perform CRUD (Create, Read, Update, Delete) operations.
- Implement searching and updating of records.
- Develop a menu-driven application.

---

## ✨ Features
- Add new student records.
- Display all student records.
- Search student by Roll Number.
- Update student details.
- Delete student records.
- Menu-driven interface.
- User-friendly input and output.
