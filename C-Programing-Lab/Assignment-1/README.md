# 📊 Student Marks Analyzer

## 📌 Project Overview
Student Marks Analyzer is a menu-driven C program that helps manage and analyze the marks of multiple students. It allows users to store student marks, display them, calculate statistical values, search for a student's marks, and sort marks in ascending order.

This project demonstrates the use of fundamental C programming concepts such as arrays, loops, functions, searching, and sorting.

---

## 🎯 Objectives
- Store marks of N students.
- Display all student marks.
- Find the highest marks.
- Find the lowest marks.
- Calculate total marks.
- Calculate average marks.
- Search marks of a particular student.
- Sort marks in ascending order.

---

## 🛠️ Features
- Enter marks for N students.
- Display all stored marks.
- Calculate highest marks.
- Calculate lowest marks.
- Calculate total marks.
- Calculate average marks.
- Search student's marks using linear search.
- Sort marks in ascending order using Bubble Sort.
- Menu-driven interface.
