# Dynamic Array Management

## 📌 Project Overview
Dynamic Array Management is a C language project that demonstrates the concept of **Dynamic Memory Allocation** using `malloc()`, `realloc()`, and `free()`. The program dynamically stores student marks, displays them, calculates statistics, and releases allocated memory properly.

This project helps in understanding pointers, arrays, and efficient memory management in C.

---

## 🎯 Objectives
- Learn Dynamic Memory Allocation in C.
- Store student marks using `malloc()`.
- Display all stored marks.
- Calculate statistics such as:
  - Total Marks
  - Average Marks
  - Highest Marks
  - Lowest Marks
- Release allocated memory using `free()`.

---

## ✨ Features
- Dynamic memory allocation using `malloc()`.
- User-defined number of student marks.
- Input and display marks.
- Calculate:
  - Total
  - Average
  - Maximum
  - Minimum
