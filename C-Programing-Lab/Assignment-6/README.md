# Inventory Management System

## 📌 Project Overview
The Inventory Management System is a menu-driven C program developed to manage product records efficiently. It uses **structures**, **arrays**, **functions**, **sorting**, and **searching** techniques to perform inventory operations. Users can add, display, search, update, sort, and calculate the total inventory value of products.

This project helps students understand how real-world inventory systems are implemented using core C programming concepts.

---

## 🎯 Objectives
- Store product information using structures.
- Add, display, search, update, and delete product records.
- Sort products based on price.
- Calculate the total inventory value.
- Learn practical applications of arrays and structures.

---

## ✨ Features
- Add new products.
- Display all products.
- Search product by Product ID.
- Update product details.
- Delete product records.
- Sort products by price (Ascending Order).
- Calculate total inventory value.
- Menu-driven interface.
- User-friendly console output.
