# 💼 Assignment-08: Employee Payroll System

## 📌 Objective
The objective of this assignment is to develop a menu-driven Employee Payroll System using C language. The program stores employee details and calculates salary components such as HRA, DA, Tax, Gross Salary, and Net Salary. It also provides employee salary reports and sorting functionality.

---

## 📝 Problem Statement

Develop an Employee Payroll System that accepts employee information including Employee ID, Name, Basic Salary, and calculates:

- HRA (House Rent Allowance)
- DA (Dearness Allowance)
- Tax
- Gross Salary
- Net Salary

The system should also:

- Add employee records
- Display payroll details
- Search employee records
- Sort employees based on salary

The program should be implemented using **Structures, Arrays, Functions, and Conditional Statements**.

---

## 🛠️ Concepts Used

- Structures
- Arrays
- Functions
- Conditional Statements
- Salary Calculations
- Searching
- Sorting
- Menu-Driven Programming

---

## 📋 Algorithm

1. Start the program.
2. Display the main menu.
3. Accept the user's choice.
4. Perform the selected operation:
   - Add Employee
   - Display Payroll Report
   - Search Employee
   - Sort Employees by Salary
5. Calculate:
   - HRA
   - DA
   - Gross Salary
   - Tax
   - Net Salary
6. Repeat until Exit is selected.
7. Terminate the program.

---

## ⏱️ Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add Employee | O(1) |
| Display Employees | O(n) |
| Search Employee | O(n) |
| Salary Calculation | O(1) |
| Sort Employees | O(n²) *(Bubble Sort)* |

---

## 💾 Space Complexity

**O(n)**

where **n** is the number of employee records.
