# 🧰 String Utility Toolkit

## 📌 Project Overview
String Utility Toolkit is a menu-driven C program that performs various string operations. It enables users to input a string and perform common operations such as finding the string length, reversing the string, checking whether it is a palindrome, counting vowels, digits, spaces, and displaying the frequency of each character.

This project helps students understand string manipulation and strengthens their knowledge of arrays, loops, and functions in C programming.

---

## 🎯 Objectives
- Accept a string from the user.
- Find the length of the string.
- Reverse the string.
- Check whether the string is a palindrome.
- Count vowels in the string.
- Count digits in the string.
- Count spaces in the string.
- Display the frequency of each character.
